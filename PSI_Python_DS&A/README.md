## Manual
### Install

    make build platform="LINUX" OR "WINDOWS" OR "MAC"
    make install platform="LINUX" OR "WINDOWS" OR "MAC"

    or

    Windows:
        install.bat

    Linux & Mac:
        install.sh

### Uninstall

    make uninstall platform="LINUX" OR "WINDOWS" OR "MAC"
 
    or

    Windows:
        uninstall.bat

    Linux & Mac:
        uninstall.sh

### Test
    python psi_test.py
    
    or
    cd Tools
    ./psi_test


### Benchmark
    python psi_benchmark.py
    
    or
    
    cd Benchmark
    ./psi_benchmark
